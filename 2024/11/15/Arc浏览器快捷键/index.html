Arc浏览器快捷键

发表于2024-11-15|更新于2024-11-25|杂项

Arc 的标签页

1️⃣ 标签页是浏览的重要组成部分，它允许你同时打开多个网页。不过，与传统浏览器不同的是，Arc 采用了垂直排列标签页的方式，这可以说使打开大量标签页比采用水平布局时更易于管理。下面列出了在 Arc 中管理标签页最有用的快捷方式，以充分利用 Arc 独特的标签页管理功能，包括归档、固定和清除标签页。

Command + T 新建标签页
Command+ W 关闭标签页
Command + Shift + T 重新打开关闭的标签页
Command + { 上一个标签页
Command + Option + ↑ 前一个标签页（备选）
Command + Option + ↓ 下一个标签页
Command + } 下一个标签页（备选）
Command + Shift + K 清除未固定的标签页
Command + D 固定/取消固定标签页

Arc的空间（Space）功能

2️⃣ 空间是 Arc 浏览器的独特功能之一，它允许你将标签分组到单独的工作空间。如果你想按主题或项目对标签页进行分组，这将非常有用，因为这样可以让你的工作空间更有条理、更整洁，从而有助于保持注意力和工作效率。每个空间都可以有自己的名称、主题甚至图标。通过点击侧边栏空间切换器菜单中的空间图标，或使用以下键盘快捷键，你可以快速切换空间。

Arc 的空间管理快捷键 功能描述
Command + Option + → 下一个空间
Command + Option + ← 上一个空间
Control + 1 前往空间 1

分栏模式(Split View)

3️⃣ Arc中的独特功能之一是分栏模式。它允许您在同一窗口中并排查看两个网页。它非常适合多任务处理、研究和比较任务。

为了帮助您充分利用此功能，Arc提供了一些键盘快捷键，使管理您的Split View窗口变得容易。

分栏模式快捷键 功能描述
Control + Shift + ] 切换到下一个分栏模式
Control + Shift + [ 切换到前一个分栏模式
Control + Shift + = 添加分栏模式（最多4个）
Control + Shift + – 删除现有的分栏模式
Control + Shift + # 切换到分栏模式＃（第 n 个）

Arc 自带的笔记功能(Arc Note)

4️⃣ Arc 的另一项独特内置功能是笔记，您可以直接在浏览器中创建和保存笔记，而无需使用外部应用程序。你可以为任何事情创建笔记，从记下快速想法到记录待办事项、研究笔记等。与标签页和网页类似，你也可以将笔记整理到文件夹和空间中。让我们来看看管理笔记的一些快捷方式。

Arc 笔记快捷键 功能描述
Control + N 新建笔记
Control + Option + N 在分栏模式新建笔记

Arc的页面导航

5️⃣ 使用键盘快捷键浏览网页可以节省时间，而且比使用鼠标或触控板更快、更有效。以下是 Arc 中用于高效页面导航的一些键盘快捷键。

页面导航快捷键 功能描述
Command + + 放大页面
Command + – 缩小页面
Command + 0 缩放至实际大小
Command + F 在页面上查找文本
Command + G 查找下一个匹配结果
Command + Shift + G 查找上一个匹配结果
Command + Option + F 查找并替换文本
Command + J 跳转到所选内容
Command + ← 后退一页
Command + → 前进一页（备选方式）
Command + [ 后退一页（备选方式）
Command + ] 前进到下一页
Command + Control + F 切换全屏模式开启/关闭

Arc 的页面控制

6️⃣ 这些控制功能对于需要操作和测试网页的高级用户和网页开发人员非常重要，例如屏幕截图、复制当前页面的URL、打开开发者工具以检查和编辑页面的HTML和CSS等等。

类似地，使用键盘快捷键可以节省时间，减少浏览菜单和设置的需要。下面让我们看一些这些键盘快捷键。

页面控制快捷键 功能描述
Command + Option + I 检查元素
Command + Option + C 检查元素
Command + Option + J JavaScript控制台
Control + C 光标选择器
Command + Shift + S 另存为…
Command + P 打印
Command + Option + Shift + V 粘贴并匹配样式
Command + Shift + 2 屏幕截图
Command + R 刷新页面
Command + Shift + R 强制刷新页面
Command + . 停止加载
Command + Shift + C 复制当前页面的网页
Command + Shift + Option + C 将当前页面的网页复制为Markdown格式

Arc 的浏览器窗口

7️⃣ 这些快捷键允许您执行常见操作，例如打开新窗口、最小化或退出浏览器。对于大多数人来说，它们与其他浏览器中的快捷键基本相同。因此，如果您之前已经使用过这些快捷键，您应该会立刻感到熟悉。

浏览器窗口管理快捷键 功能描述
Command + N 新建浏览器窗口
Command + Shift + N 新建隐身浏览器窗口
Command + W 关闭浏览器窗口
Command + H 隐藏 Arc 浏览器
Command + Shift + H 隐藏除 Arc 浏览器外的所有内容
Command + M 最小化浏览器窗口
Command + Q 退出 Arc 浏览器

Little Arc（小浏览器）

8️⃣ “Little Arc”是Arc浏览器中的一项独特功能，它打开一个无边框的窗口，一次只包含一个页面，非常适合快速查找；在不中断主要工作流程或窗口的情况下快速轻松地访问信息。

通过下面的键盘快捷键，您可以充分利用这一功能，进一步提高您的生产力。

Little Arc 快捷键 功能描述
Command + Option + N 新建 Little Arc 窗口
Command + O 打开 Little Arc 标签页
Command + Shift + O 显示 Little Arc 的 “打开方式” 菜单

Arc其他快捷键

9️⃣ Arc浏览器还配备了其他实用的快捷键。其中一些快捷键可能不太常用，但我相信随着您对其进行更多探索，您会发现它们在改善特定任务和工作流程方面非常方便。

其他快捷键 功能描述
Command + 逗号 进入偏好设置
Command + Shift + J 查看下载内容
Command + Y 查看历史记录
Command + Shift + L 打开库
Command + E 打开扩展
Command + Option + U 查看页面源代码
Command + Shift + R 进入阅读模式
Command + L 启动命令栏
Command + S 显示/隐藏侧边栏
Command + # 转到侧边栏项目 #（1、2等）
Control + Shift + E 新建画布
